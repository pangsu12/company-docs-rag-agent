\# Company Docs RAG Agent



\## 1. 프로젝트 소개



이 프로젝트는 회사 내부 문서, 회의록, 정책 문서, FAQ 등을 기반으로 사용자의 질문에 답변하는 LLM/RAG 기반 AI Agent 서비스입니다.



단순 챗봇이 아니라, 문서 검색, 출처 기반 답변 생성, 문서 요약, FAQ 생성, 액션 아이템 추출 등 실제 업무 자동화에 필요한 기능을 목표로 합니다.



\## 2. 개발 배경



저는 모바일 앱과 서비스 개발 경험을 바탕으로, LLM/RAG 기반 AI Agent를 실제 서비스와 업무 자동화에 적용하는 AI 엔지니어를 목표로 하고 있습니다.



기존에 React Native와 Supabase를 활용한 앱 개발 경험이 있으며, 이 프로젝트에서는 Python, 문서 처리, 검색, LLM API, FastAPI, AI Agent 기능을 단계적으로 구현하고자 합니다.



\## 3. 핵심 기능



\- 회사 문서 업로드

\- 문서 chunk 분할

\- 문서 검색

\- 사용자 질문 입력

\- 관련 문서 기반 답변 생성

\- 답변 출처 표시

\- 질문 기록 저장

\- 문서 요약

\- FAQ 자동 생성

\- 회의록 액션 아이템 추출

\- 이메일 초안 작성



\## 4. 기술 스택
